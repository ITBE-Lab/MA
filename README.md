
## Getting started

    git clone https://github.com/ItBeLab/ma
    cd ma
    make
    ./bwa index ref.fa
    ./bwa mem ref.fa read-se.fq.gz | gzip -3 > aln-se.sam.gz
    ./bwa mem ref.fa read1.fq read2.fq | gzip -3 > aln-pe.sam.gz
